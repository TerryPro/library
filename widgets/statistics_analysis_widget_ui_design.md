### 1. 标题栏设计

**设计说明**:

#### 列选择器
```
□ 选择要分析的列 (4/5 已选择)

├─□ 湿度 (数值)
├─□ 电压 (数值)
```

**设计说明**:
- 显示列名和数据类型
- 实时显示选择数量
- 支持全选/取消全选

#### 图表类型选择器

```
图表类型:
```

**设计说明**:

## 📋 设计说明

### 设计原则
1. **简洁直观**: 避免过度复杂的界面元素
2. **功能聚焦**: 突出核心统计分析功能
3. **响应迅速**: 快速的参数调整和图表更新
4. **用户友好**: 清晰的标签和帮助信息

### 用户流程
1. 打开组件 → 自动检测数值列
2. 选择要分析的列 → 系统自动计算统计量
3. 选择图表类型 → 实时渲染对应图表
4. 调整配置选项 → 即时预览效果
5. 导出结果 → 保存为所需格式

这个UI设计图为开发团队提供了清晰的视觉参考，确保最终实现的产品符合设计预期。
